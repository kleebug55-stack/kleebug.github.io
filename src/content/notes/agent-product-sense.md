Agent 产品感不是来自“让它更聪明”的口号，而是来自对边界的敏感。

## 暂时的判断

- 用户要的不是自主性本身，而是可靠地省心。
- Agent 每多一步自动行动，就需要多一层可解释和可撤销。
- 好的 Agent 体验应该让用户知道：它正在做什么、为什么这么做、哪里需要我介入。

## 还在想的问题

什么时候应该让 Agent 主动追问？什么时候应该先给出一个不完美但可修改的草稿？这两个选择背后，其实是对风险、信任和用户心智负担的判断。
